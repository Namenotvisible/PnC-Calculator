# Combinatorial Computation Engine & Calculator

## Overview
A lightweight, interactive command-line application engineered in C to calculate complex mathematical probabilities, permutations, and combinations. The application provides a straightforward, menu-driven interface to execute core mathematical computations with dynamic user inputs.

## Core Features & Logic

* **Recursive Computation:** Utilizes deep recursive functions to calculate factorials, establishing the foundational mathematics for combinations and permutations.
* **Interactive CLI:** Features a persistent, menu-driven Command Line Interface (CLI) that gracefully handles continuous user requests until manually terminated.
* **Probability Engine:** Calculates fractional probabilities based on favorable versus total outcomes, utilizing type-casting for precise floating-point outputs.
* **Combinatorial Logic:** Accurately computes $C(n, r)$ and $P(n, r)$ formulas directly from user-defined constraints.

## Tech Stack
* **Language:** C
* **Paradigms:** Recursion, Command-Line Interface Design, Mathematical Modeling
